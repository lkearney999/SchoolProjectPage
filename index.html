Project Report - World Design


Introduction


For my project I worked with Luca doing world design in the hammer world editor. Using the software we aimed to create a basic world for Counter-Strike:Global Offensive. The world was aimed to create a fair one on one tournament so we designed the word so it was A symmetrical in order to ensure a balanced competition.


Method


We started out with an basic draft on paper of what the map would look like, which Luca then drafted into illustrator so we would have a plan to follow. We then set about designing the map in the Hammer Editor (An old port of the source SDK). We encounter few problems as we had both used the software before so we were able to quickly construct the map after we overcame a few installation problems. apart from the installation issues the only barrier we encountered was with making the map look visually appealing with little to no time to fully design textures, to overcome this we used the default assets but mixed up things like decals, props and foliage to break originally bleak and grey textures of the concrete.


Results


Using the software we were able to design a functioning map with two players spawns, functioning lighting, sound effects, vectorized terrain and two large peaking ramps to encourage aggressive play. One of the better aspects of the map includes a car crash into the concrete were the car is smashed into the concrete leaving a visible crash on the ramp using decals and the car is well tilted so the crash looks to have aged as the terrain conceals part of the car well and the surrounding trash adds to the dilapidated look.
